# 🌸 Personal Portfolio Website

Website portofolio pribadi yang dibuat menggunakan HTML, CSS, dan JavaScript. Website ini berisi informasi tentang profil, keterampilan, proyek yang pernah dikerjakan, serta informasi kontak dalam tampilan yang responsif dan mudah digunakan.

## 📌 Fitur

* Home
* About Me
* Skills
* Portfolio
* Preview proyek menggunakan video
* Contact
* Responsive Design

## 🛠️ Teknologi

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Fonts

## 📁 Struktur Folder

```text
├── index.html
├── style.css
├── script.js
├── img/
├── video/
└── README.md
```
